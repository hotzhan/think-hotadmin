特点

自由扩展的组件生态

基于最新 ThinkPHP 框架开发，遵循并使用 Composer 标准管理依赖组件，可自由安装各种开源组件及插件生态程序。

标准化文件存储引擎

支持本地服务器存储、自建 Alist 存储服务器、多种云空间存储，基于文件 HASH 实现文件秒传，同时节省服务器空间。

标准化文件存储引擎

支持本地服务器存储、自建 Alist 存储服务器、多种云空间存储，基于文件 HASH 实现文件秒传，同时节省服务器空间。

独立进程异步执行任务

兼容多平台动态创建 PHP 进程，并列启动多个独立任务进程同时处理大数据或长时性任务，并实时显示执行进度及结果。
