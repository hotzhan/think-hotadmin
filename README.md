# think-hotadmin
基于thinkphp8/adminlte3.2后台管理系统

## 项目clone到本地
```shell
git clone https://github.com/hotzhan/think-hotadmin.git
```

## 安装项目依赖
```shell
composer install
```

## 需要修改的配置文件
* config/mail.php 邮件服务器配置（HotAdmi专用）
* config/sms.php 短信网关配置（HotAdmin专用）
* 其它配置文件参考ThinkPHP6/8的文档即可

## 数据库文件
sql.sql导入到自己的网站mysql数据库
